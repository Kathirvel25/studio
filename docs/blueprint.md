# **App Name**: StudyStreak AI

## Core Features:

- Student Profile Setup: Allow students to input their name, class/year, preferred study timings, and subjects.
- Task & Syllabus Input: Enable students to enter exams, assignments, project deadlines, meetings, classes, and chapter-wise syllabus details.
- Smart Schedule Generator: Automatically generate daily and weekly study schedules, prioritizing urgent tasks and adjusting schedules based on deadlines using rule-based logic.
- Dashboard: Display todayâ€™s tasks, weekly goals, progress graphs, completion percentage, and upcoming deadlines in a student-friendly dashboard.
- Notifications & Reminders: Send study reminders, deadline alerts, break reminders, and daily consistency reminders using Firebase Cloud Messaging.
- Streak Tracking: Track daily study streaks and award streak badges for consistency (e.g., 7 days, 30 days).
- AI Weekly Smart Feedback: Generate personalized weekly reports using a Gemini tool that analyzes completed tasks, missed tasks, study time, and difficulty levels, offering targeted advice such as, 'You completed 68% of your weekly plan. You should focus more on Math next week.'

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calmness and focus, promoting a conducive environment for studying.
- Background color: Light gray (#F0F4F7), nearly desaturated blue, provides a neutral backdrop that reduces eye strain and keeps the interface clean.
- Accent color: Muted purple (#B39DDB) to add subtle visual interest and highlight important interactive elements without distracting from the primary content.
- Body and headline font: 'PT Sans', a humanist sans-serif, suitable for both headlines and body text.
- Use minimalist icons to represent different tasks, subjects, and reminders, ensuring clarity and ease of use.
- Design a clean and intuitive dashboard layout with progress bars and clearly defined sections for tasks, goals, and deadlines.
- Implement subtle animations for task completion and streak achievements to provide positive feedback and engagement.
